# 🎨 Portfolio Professionnel - Thomas Graphiste

Bienvenue sur le projet de mon tout premier site web portfolio ! Ce site a été développé de A à Z en HTML et CSS.

L'objectif de ce projet est de présenter mes compétences, mes articles et mes travaux, tout en appliquant les bonnes pratiques du développement web.

## 🚀 Ce que j'ai mis en place sur ce site

### 1. Une structure propre (HTML)
Le site utilise le système de boîtes (balises) pour être bien organisé :
* Un menu de navigation (`<nav>`).
* Une introduction claire (`<header>`).
* Des sections séparées pour le Blog, les Projets et le Contact (`<section>`).

### 2. Une mise en page moderne (CSS Flexbox)
L'agencement des éléments (comme mettre les articles de blog côte à côte ou aligner le formulaire) a été réalisé avec la propriété CSS `Flexbox`.

### 3. Un site rapide et optimisé
Pour que le site charge très vite, j'ai utilisé la balise `<picture>`. Elle permet au navigateur de choisir des images très légères (format `.webp`), tout en gardant des images classiques (`.png`) en secours pour les vieux navigateurs.

### 4. L'adaptation aux téléphones (Responsive)
Le design s'adapte automatiquement sur les petits écrans (téléphones) : 
* Les éléments qui étaient côte à côte se placent les uns sous les autres.
* Le menu classique se transforme en bouton de menu mobile.

### 5. Une bonne expérience utilisateur
* Ajout d'un petit logo dans l'onglet du navigateur internet (le `favicon`).
* Possibilité pour les visiteurs de télécharger mon CV au format PDF d'un simple clic grâce à l'attribut `download`.

## 🛠️ Comment ouvrir ce site chez vous ?

1. Téléchargez les fichiers de ce projet.
2. Ouvrez le dossier sur votre ordinateur.
3. Double-cliquez sur le fichier `index.html` : il va s'ouvrir tout seul dans votre navigateur internet !
